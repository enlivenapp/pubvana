Comments

Pubvana includes a built-in comment system with a moderation queue.

Comment Moderation Queue

New comments land in the moderation queue at Comments → Pending. For each comment you can:

  • Approve — publishes the comment under the post
  • Spam — marks as spam and hides
  • Trash — soft-deletes (recoverable)
  • Delete — permanently removes

Moderation Settings

In Settings → General, the Comment Moderation toggle controls behaviour:

  • Require approval (default) — all new comments go into the queue
  • Auto-approve — comments appear immediately

Disabling Comments Site-Wide

In Settings → General, uncheck Enable Comments to disable the comment form on all posts globally. Existing comments remain visible.
