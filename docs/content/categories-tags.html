Categories & Tags

Use categories and tags to organise your posts and help readers browse related content.

Categories

Categories are the primary taxonomy — broad topic buckets. Go to Posts → Categories to manage them.

  • Name — displayed in menus and post listings
  • Slug — used in the URL (auto-generated)
  • Description — optional; shown on category archive pages

Assigning Categories to Posts

On the post editor, check one or more categories in the Categories panel. A post can belong to multiple categories.

Tags

Tags are more granular labels. In the post editor, type tags in the Tags field separated by commas. New tags are created on the fly — no need to pre-create them.

Post Count Display

The Categories and Tags admin screens show the post count next to each item. Zero-count tags and categories are kept but not shown in front-end tag clouds unless you add them manually.
