Media Library

The Media Library is where all uploaded images and documents are stored and managed.

Uploading Files

Go to Media → Upload. Drag files onto the upload area or click to browse. Supported types include JPEG, PNG, GIF, WebP, and PDF. Uploaded images are automatically converted to WebP format to reduce file size.

Inserting Media into Posts

In the post editor, click the Add Media button (image icon in the toolbar). The media picker opens — click any image to insert it at the cursor position. You can resize, align, or add a caption directly in the editor.

Managing Files

The media grid at Media → Library shows all uploads. Click any item to:

  • View its URL (for manual linking)
  • See which posts use it
  • Delete it — note that deleting removes the file from all posts where it is used
Before deleting Check that the file is not in use. The media detail panel shows “Used in N posts” — delete only when that count is 0.
