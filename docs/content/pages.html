Pages

Pages are static content pieces — About, Contact, Privacy Policy — that sit outside the blog feed.

Pages vs Posts

PostsPages
Appear in blog feedYesNo
Have categories/tagsYesNo
Date-stampedYesNo
Can be front pageNoYes

Creating a Page

Go to Pages → New Page. Enter a title, write the content, and click Save Page. The slug is auto-generated from the title.

Setting a Static Front Page

By default, the homepage shows the blog feed. To display a page instead, go to Settings → General and set Front Page to the desired page.

System Pages

Some pages are marked as system pages and cannot be deleted. You can still edit their content.

Page Order

Each page has a Sort Order field. Lower numbers appear first in navigation menus that auto-populate from pages.
