Navigation Menus

Navigation menus control the links that appear in your theme’s header and footer.

Menu Types

  • Primary menu — displayed in the site header (main navigation)
  • Footer menu — displayed in the site footer

Adding Items

  1. Go to Design → Navigation
  2. Select the menu (Primary or Footer)
  3. Click Add Item
  4. Choose the type: Page, Post, Category, or Custom URL
  5. Set the label and target URL
  6. Optionally tick Open in new tab
  7. Click Save Menu

Removing Items

Click the × icon on any menu item to remove it, then save.

Reordering

Drag items by their handle to reorder. Drag an item slightly to the right to make it a child (sub-menu) of the item above it.

Sub-Menus

Indent an item under a parent to create a dropdown. Most themes support one level of sub-menus.
