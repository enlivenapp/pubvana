Themes

Themes control the visual appearance of your public-facing site. Changing a theme is instant and non-destructive.

Browsing Available Themes

Go to Design → Themes. Installed themes are shown as cards with a preview screenshot, name, and version.

Activating a Theme

Click Activate on the theme you want. The front end switches immediately. Only one theme can be active at a time.

Theme Options

After activating, click Customise to open the theme options panel. Common options include:

  • Show sidebar — toggle the sidebar widget area
  • Footer copyright — text displayed in the footer
  • Hero tagline — large text on the homepage hero (theme-dependent)
  • Accent colour — hex colour code for the primary accent (e.g. #f59e0b)

Premium Themes Premium

Premium themes are distributed through the Marketplace. When you install a premium theme you will be prompted for your licence key, which is validated against pubvana.net and tied to your domain.

Installing a Theme from the Marketplace

  1. Go to Tools → Marketplace
  2. Click Themes to filter
  3. Click Install on the theme you want
  4. For premium themes, enter your licence key when prompted
  5. After installation the theme appears in Design → Themes
