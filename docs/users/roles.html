Roles & Permissions

Pubvana uses five roles to control what each user can do in the admin panel.

Role Overview

RoleCapabilities
SubscriberRead published content, submit comments, manage own profile
AuthorCreate and edit own posts, upload media
EditorCreate and edit all posts and pages, manage comments, categories, tags
AdminFull access: themes, widgets, settings, marketplace — except managing other admins
SuperadminFull access including user deletion, role changes, and licence management

Creating a User Account

  1. Go to Users → New User
  2. Enter email, display name, and a temporary password
  3. Assign a role
  4. Click Create User

The user receives a welcome email with login instructions (requires outgoing mail to be configured).

Activating / Deactivating Accounts

On the Users list, click the status badge next to a user to toggle between Active and Inactive. Inactive users cannot log in but their content is preserved.
