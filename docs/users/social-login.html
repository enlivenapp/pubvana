Social Login

Allow visitors to sign in with their Google or Facebook account instead of creating a separate password.

Enabling Google Login

  1. Go to Google Cloud Console
  2. Create a new project (or select an existing one)
  3. Navigate to APIs & Services → Credentials
  4. Click Create Credentials → OAuth client ID
  5. Application type: Web application
  6. Add your redirect URI: https://yourdomain.com/auth/google/callback
  7. Copy the Client ID and Client Secret
  8. Paste them into Settings → Social → Google in your Pubvana admin

Enabling Facebook Login

  1. Go to Meta for Developers
  2. Create a new app (Consumer type)
  3. Add the Facebook Login product
  4. Add the redirect URI: https://yourdomain.com/auth/facebook/callback
  5. Copy the App ID and App Secret
  6. Paste them into Settings → Social → Facebook in your Pubvana admin

How Users See It

Once a provider is configured, the login and registration pages display a Sign in with Google or Sign in with Facebook button. Users who sign in via social are created with the Subscriber role by default.
