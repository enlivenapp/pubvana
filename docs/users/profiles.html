Author Profiles

Author profiles add a bio card below posts, helping readers connect with your writers.

Editing Your Profile

Go to Users → Your Profile (or Users → All Users → Edit for another account).

  • Display Name — shown on posts and the author card
  • Bio — short paragraph about the author
  • Avatar — upload a profile photo (JPG/PNG, square recommended)

Social Profile Links

The profile page includes fields for:

  • Personal website URL
  • Twitter / X handle
  • Facebook profile URL
  • LinkedIn profile URL

These appear as icon links in the author card beneath each post.

Author Card on Posts

When a post is published by a user with a completed profile, a card automatically appears below the post content showing the author’s avatar, display name, bio, and social links. The Author Bio widget also renders this card — see Widgets.
