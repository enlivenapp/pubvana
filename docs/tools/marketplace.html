Marketplace

The Marketplace lets you browse and install themes and widgets — both free and premium — directly from the admin.

Browsing

Go to Tools → Marketplace. Items are fetched from pubvana.net and cached for one hour. Use the filter buttons to show All, Themes, or Widgets.

Installing a Free Item

  1. Find the item and click Install
  2. Pubvana downloads and extracts the package automatically
  3. For themes, it appears in Design → Themes; for widgets, in Design → Widgets

Installing a Premium Item Premium

  1. Purchase the item at pubvana.net/store and copy your licence key
  2. Click Install on the item in the Marketplace
  3. Enter your licence key when prompted
  4. The key is validated against pubvana.net and stored for this domain

Checking for Updates

Installed items with a newer version available show an Update button. Click it to download and apply the update. Your configuration and data are preserved.

Cache Refresh

The marketplace listing is cached for one hour. If you’ve just purchased a new item and it’s not showing up, click Refresh Cache to force a fresh fetch from pubvana.net.
