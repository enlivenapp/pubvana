WordPress Import

Migrate your existing WordPress site to Pubvana using the WXR (WordPress eXtended RSS) export format.

Step 1 — Export from WordPress

  1. In your WordPress admin, go to Tools → Export
  2. Select All content
  3. Click Download Export File
  4. You’ll receive an XML file (the WXR)

Step 2 — Upload the WXR File

In Pubvana, go to Tools → Import → WordPress. Upload the XML file (maximum 50 MB). For larger exports, use the CLI method below.

Dry-Run Preview

Tick Dry run (preview only) before importing. Pubvana will parse the file and show you a summary of what will be imported without actually creating any records. This lets you verify counts before committing.

What Gets Imported

  • Posts — title, content, excerpt, date, status, featured image URL
  • Pages — title, content, date, status
  • Categories — name, slug, description
  • Tags — name, slug
  • Authors — created as Subscriber accounts (set passwords separately)
  • Comments — approved comments with author name and email

CLI Alternative

For large files or automated pipelines, use the Spark command:

php spark wp:import /path/to/export.xml

Add --dry-run for a preview without importing.
