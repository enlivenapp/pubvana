CLI Commands

Pubvana includes several php spark commands for automation and maintenance.

Available Commands

CommandDescription
php spark wp:import <file> Import a WordPress WXR export file. Add --dry-run for preview mode.
php spark posts:publish Publish all scheduled posts whose publish date has passed. Run via cron.
php spark links:check Scan all post and page content for broken external links. Results appear in Admin → Broken Links.
php spark marketplace:revalidate Re-validate all installed premium licence keys against pubvana.net.
php spark db:seed PubvanaSeeder Seed the database with default data (run once during installation).

Cron Job: Post Scheduling

To enable automatic post publishing, add this cron job (runs every minute):

* * * * * /usr/bin/php /path/to/pubvana/spark posts:publish >> /dev/null 2>&1

Cron Job: Broken Link Checker

A weekly scan is recommended (Sundays at 2 AM here, adjust to suit):

0 2 * * 0 /usr/bin/php /path/to/pubvana/spark links:check >> /dev/null 2>&1
Path to PHP Find your PHP binary path with which php or which php8.1 if you have multiple versions. Use the full absolute path in cron jobs.
