Social Sharing

Auto-post new articles to Twitter/X and Facebook when they are published.

Twitter / X

You need a Twitter Developer account with a project and app that has Read and Write permissions.

  1. Go to developer.twitter.com
  2. Create a project and app
  3. Under Keys and Tokens, generate:
  • API Key (Consumer Key)
  • API Key Secret (Consumer Secret)
  • Access Token
  • Access Token Secret

Paste all four values into Settings → Social Sharing → Twitter.

Facebook

  1. Create a Facebook Page for your site
  2. Go to developers.facebook.com and create an app
  3. Add the Pages API product and generate a Page Access Token
  4. Enter your Page ID and Page Access Token in Settings → Social Sharing → Facebook

Enabling Auto-Share

Tick Enable auto-share on publish. From that point, any post published with the Share on publish checkbox checked will be automatically posted to the configured networks.

Per-Post Control

In the post editor, expand the Social panel and toggle Share on publish on or off for individual posts.
