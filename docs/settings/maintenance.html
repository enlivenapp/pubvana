Maintenance Mode

Maintenance mode shows a friendly “under maintenance” page to public visitors while you make changes.

Toggling Maintenance Mode

Go to Settings → General and toggle Maintenance Mode, then click Save Settings. The change takes effect immediately.

Who Can Still Access the Site

While maintenance mode is on:

  • Admins and Superadmins can access all pages normally
  • Any logged-in user can read the site
  • Public (logged-out) visitors see the maintenance page
  • The login and register pages remain accessible so admins can sign in

Typical Use Cases

  • Applying major design changes or a new theme
  • Performing a bulk WordPress import
  • Running database migrations or updates
  • Writing and scheduling a batch of posts before launch
Remember to turn it off It’s easy to forget maintenance mode is active. Check your site in a private/incognito window after making changes to confirm public visitors can access it.
