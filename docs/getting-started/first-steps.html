First Steps

You’ve installed Pubvana — here’s how to get your site set up in under 10 minutes.

1 — Admin Dashboard Tour

After logging in at /admin you’ll see the dashboard with:

  • Stats bar — total posts, pages, comments, users
  • Recent posts — quick links to edit your latest content
  • Quick actions — New Post, New Page, Manage Widgets
  • Left sidebar — full navigation to all admin sections

2 — Set Your Site Name & Tagline

Go to Settings → General. Enter your site name and tagline — these appear in the browser title, header, and RSS feed. Click Save Settings.

3 — Create Your First Post

  1. Go to Posts → New Post
  2. Enter a title — the slug is auto-generated
  3. Write your content in the rich-text editor
  4. Assign a category and add tags (comma-separated)
  5. Upload a featured image
  6. Set status to Published and click Save Post

4 — Choose a Theme

Go to Design → Themes. Click Activate on the theme you want. The change takes effect immediately.

5 — Add Widgets to the Sidebar

Go to Design → Widgets. Select the Sidebar area. Click Add Widget, choose a type, configure it, and save. Drag to reorder.

6 — Configure Navigation

Go to Design → Navigation. Add your key pages (Home, Blog, About, Contact) and click Save Menu.
