Installation

Requirements

RequirementMinimumNotes
PHP8.1+Extensions: intl, mbstring, json, mysqlnd
MySQL / MariaDB5.7 / 10.3+InnoDB engine required
Composer2.xFor dependency management
Apache / NginxAny recentmod_rewrite (Apache) or try_files (Nginx)

1 — Download

Clone the repository or download a release archive:

git clone https://github.com/your-org/pubvana.git
cd pubvana

2 — Install Dependencies

composer install --no-dev --optimize-autoloader

3 — Configure Environment

Copy the example environment file and edit it:

cp env .env

Open .env and set at minimum:

# Application
CI_ENVIRONMENT = development
app.baseURL    = 'http://yourdomain.com/'

# Database
database.default.hostname = localhost
database.default.database = your_database
database.default.username = your_user
database.default.password = your_password
database.default.DBDriver = MySQLi

4 — Create Database & Run Migrations

Create an empty MySQL database, then run:

# Core app migrations
php spark migrate

# Shield (auth) migrations
php spark migrate --namespace "CodeIgniter\Shield" --verbose

# Settings migrations
php spark migrate --namespace "CodeIgniter\Settings" --verbose
Note on Shield migrations The --verbose flag is required for Shield and Settings namespace migrations; without it they may silently skip.

5 — Seed the Database

php spark db:seed PubvanaSeeder

This creates the default admin account, sample categories, and core configuration.

6 — Web Server Setup

Point your web server's document root to the public/ directory.

Apache

The public/.htaccess file is included. Ensure mod_rewrite is enabled and AllowOverride All is set for the directory.

Nginx

location / {
    try_files $uri $uri/ /index.php$is_args$args;
}

7 — Set Permissions

chmod -R 777 writable/

8 — First Login

Open your browser to http://yourdomain.com/admin. Log in with the default credentials created by the seeder (change these immediately in Users → Edit).

Installation complete! Head to First Steps to configure your site.
