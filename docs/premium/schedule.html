Post Schedule Premium

⭐

This feature requires a Premium Core licence. Activate →

Schedule posts to publish automatically at a specific date and time.

Scheduling a Post

  1. Open the post editor
  2. In the Status dropdown, select Scheduled
  3. A date/time picker appears — choose the publish date and time
  4. Click Save Post

The post sits in a Scheduled state until the cron job publishes it.

Calendar View

Go to Admin → Schedule to see a monthly calendar showing all scheduled posts. Click any post to open it for editing.

Cron Requirement

Automatic publishing requires the following cron job to be set up on your server (runs every minute):

* * * * * /usr/bin/php /path/to/pubvana/spark posts:publish >> /dev/null 2>&1
Without the cron job Scheduled posts will not publish automatically. You can still publish them manually (see below).

Manual Publish via CLI

To publish all due scheduled posts immediately without waiting for the cron:

php spark posts:publish
