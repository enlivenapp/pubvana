SEO Settings Premium

⭐

This feature requires a Premium Core licence. Activate →

Go to Settings → SEO to configure search-engine optimisation options.

Site Meta Description

The default meta description used when a post or page doesn’t have its own. Keep it under 160 characters.

Google Analytics

Enter your Tracking ID (format: G-XXXXXXXXXX for GA4 or UA-XXXXXXXX-X for Universal Analytics). Pubvana inserts the tracking snippet into every page automatically.

XML Sitemap

Enable XML Sitemap to generate /sitemap.xml automatically. The sitemap includes all published posts and pages. Submit the URL to Google Search Console and Bing Webmaster Tools.

News Sitemap

Enable News Sitemap to generate a Google News-compatible sitemap at /sitemap-news.xml. Useful if you publish time-sensitive news content. Requires the XML sitemap to be enabled first.
