Activity Log Premium

⭐

This feature requires a Premium Core licence. Activate →

The Activity Log records every significant admin action, giving you a full audit trail of who did what and when.

What Gets Logged

  • Posts & Pages — created, edited, published, deleted
  • Users — created, role changed, activated, deactivated
  • Themes — activated, installed, options saved
  • Widgets — added, reordered, deleted
  • Settings — any settings page saved
  • Marketplace — items installed or updated

Viewing the Log

Go to Admin → Activity Log. Each entry shows:

  • User — who performed the action
  • Timestamp — exact date and time
  • Description — a plain-English summary of what changed

Filtering

Use the Filter by type dropdown to narrow the log to a specific category (e.g. show only Posts activity or only Settings changes).
