Affiliate Links Premium

⭐

This feature requires a Premium Core licence. Activate →

Create short, trackable redirect links at /go/{slug} for affiliate programmes, sponsored content, or campaign tracking.

Creating a Link

  1. Go to Admin → Affiliate Links
  2. Click New Link
  3. Enter a Slug — the short code used in the URL (e.g. hosting-deal)
  4. Enter the Destination URL — the full affiliate URL
  5. Add an optional Note to identify the campaign
  6. Click Save

Visitors who go to https://yourdomain.com/go/hosting-deal are immediately redirected (301) to the destination URL, and the click is recorded.

Toggling Active / Inactive

Click the status toggle next to any link to enable or disable it. Inactive links return a 404 instead of redirecting.

Viewing Click History

Click View Stats next to any link to see:

  • Total click count
  • Daily click chart (last 30 days)
  • Recent click timestamps

Use Cases

  • Affiliate programme links (Amazon, ShareASale, etc.)
  • Sponsored content — track clicks to a sponsor’s site
  • Campaign tracking — compare performance of different placements
  • Clean URLs — hide long, ugly affiliate parameters from readers
