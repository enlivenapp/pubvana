Analytics Premium

⭐

This feature requires a Premium Core licence. Activate →

Built-in page view analytics — no external service required. Data is stored in your own database.

Accessing Analytics

Go to Admin → Analytics. The dashboard shows:

  • Total views for the selected period (7 / 30 / 90 days)
  • Top posts — ranked bar chart of most-viewed content
  • Referrer breakdown — where traffic is coming from
  • Daily view chart — line graph of views over time

View Periods

Use the period selector at the top of the analytics page to switch between 7-day, 30-day, and 90-day windows.

How Tracking Works

Pubvana records a page view each time a post or page is loaded on the front end. Bot traffic is filtered using a basic user-agent check. Views are stored in the page_views table in your database — no cookies are set and no data leaves your server.

JSON Data Endpoint

For custom integrations, the analytics data is available as JSON:

GET /api/analytics/views?period=30

Returns an array of { date, views } objects for the specified period. Requires an authenticated admin session.
