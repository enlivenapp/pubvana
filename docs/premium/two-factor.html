Two-Factor Auth Premium

⭐

This feature requires a Premium Core licence. Activate →

Add an extra layer of security to your account with time-based one-time passwords (TOTP).

Setting Up 2FA

  1. Go to Users → Your Profile → Security
  2. Click Enable Two-Factor Auth
  3. A QR code is displayed — scan it with an authenticator app:
    • Google Authenticator (iOS / Android)
    • Authy (iOS / Android / Desktop)
    • 1Password, Bitwarden, or any TOTP-compatible app
  4. The app generates a 6-digit code that changes every 30 seconds
  5. Enter the current code in the Verify Code field and click Confirm
  6. 2FA is now active on your account
Save your backup codes After enabling 2FA, Pubvana shows a set of single-use backup codes. Store them somewhere safe (password manager or printed copy). You’ll need them if you lose access to your authenticator app.

Using 2FA at Login

After entering your email and password, you’ll be prompted for a 6-digit code. Open your authenticator app, find the Pubvana entry, and type the current code.

Disabling 2FA

  1. Go to Users → Your Profile → Security
  2. Click Disable Two-Factor Auth
  3. Enter your current password to confirm

A Superadmin can also disable 2FA for any user from Users → Edit User → Security.
